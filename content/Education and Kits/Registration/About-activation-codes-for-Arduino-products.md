## Products with activation codes that unlock features

The following products have activation codes that unlock additional features:

| Product                                                                                              | Feature                                                                                                                                                                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Arduino Engineering Kit<br>Arduino Engineering Kit R2                                                | The course is now free, but the activation code is still used to unlock MathWorks software. See [Activate the Engineering Kit MATLAB and Simulink license](https://support.arduino.cc/hc/en-us/articles/4406437015186-Activate-the-Engineering-Kit-MATLAB-and-Simulink-license).                                                                             |
| Arduino Oplà IoT Kit                                                                                 | Includes a code for 12 free months of the Cloud Maker Plan that can be applied during checkout.                                                                                                                                                                                                                                                              |
| Arduino Starter Kit R4                                                                               | [Register the kit](https://support.arduino.cc/hc/en-us/articles/24775562011036-Register-your-Arduino-Starter-Kit-R4) to access translations and additional chapters online.<br>The kit also includes an [activation code for the Arduino Certification exam](https://support.arduino.cc/hc/en-us/articles/4405108078354-Take-an-Arduino-Certification-exam). |
| Arduino UNO R4 WiFi (standalone or included in kit)<br>_Not available for Arduino UNO Ek एक R4 WiFi_ | Activate your UNO R4 WiFi board to get 3 months of the Arduino Cloud Maker plan. Go to [arduino.cc/start](https://www.arduino.cc/start), select UNO R4 WiFi, and follow the instructions. _Do not use the “Pick your board or kit” dropdown menu, as it does not list the UNO R4 WiFi._                                                                      |

> [!NOTE]
> To redeem the 12 free months of Maker plan for the Arduino Explore IoT Kit, please [contact us](https://www.arduino.cc/en/contact-us/).

## Products that no longer require activation

Many online courses and content for Arduino products that previously required an activation code are now available for free. If you have one of the kits listed below, you can ignore any instructions to activate it.

- Arduino Braccio ++
- Arduino CTC 101
- Arduino CTC GO! Core Module
- Arduino CTC GO! Motions Expansion Pack
- Arduino Education Starter Kit
- Arduino Engineering Kit
- Arduino Engineering Kit R2
- Arduino Explore IoT Kit*
- Arduino Greenhouse Kit
- Arduino Science Kit Physics Lab
- Arduino Science Kit Physics Lab Rev2
- Arduino Student Kit
