The Arduino IDE installation folder contains the core application executables, bundled command-line tools (such as `arduino-cli`), and default configuration files.

> [!IMPORTANT]
> Downloaded libraries, sketches, and board packages are not located in the installation folder. Please see [Find sketches, libraries, board cores, and other files on your computer](https://support.arduino.cc/hc/en-us/articles/4415103213714-Find-sketches-libraries-board-cores-and-other-files-on-your-computer) instead.

## Default installation folders

Below are the default installation paths for Arduino IDE based on your operating system and installation choice:

| Operating System | Installation Type | Default Path / Location |
| :--- | :--- | :--- |
| **Windows** | Standard | `C:\Program Files\Arduino IDE` (system install) or `%localappdata%\Programs\Arduino IDE` (user install) |
| **Windows** | Microsoft Store | `C:\Program Files\WindowsApps\ArduinoLLC.ArduinoIDE_<version>` |
| **macOS** | Standard | `/Applications/Arduino IDE.app` |
| **Linux** | AppImage | *Portable (runs directly from the downloaded `.AppImage` file)* |
| **Linux** | ZIP Archive | *User-defined folder (where you extracted the ZIP/TAR archive)* |
